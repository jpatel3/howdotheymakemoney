## Color Scheme and Typography

### Color Palette
- **Primary Color**: #3366CC (Blue) - Conveys trust, professionalism
- **Secondary Color**: #FF9900 (Orange) - Adds energy, creativity
- **Background**: #FFFFFF (White) - Clean, minimalist
- **Text**: #333333 (Dark Gray) - Easy readability
- **Accent**: #66CC99 (Mint Green) - For highlights and CTAs

### Typography
- **Headings**: Montserrat (Sans-serif) - Modern, clean
- **Body Text**: Inter (Sans-serif) - Highly readable
- **Data Visualization Labels**: Roboto Mono (Monospace) - Technical precision

### UI Components
- **Search Bar**: Prominent, rounded corners with subtle shadow
- **Cards**: White background with light shadow, rounded corners
- **Charts**: Clean, minimalist with the color palette above
- **Buttons**: Solid fill for primary actions, outlined for secondary actions

## Data Visualization Style Guide

### Revenue Breakdown Chart
- Donut chart with segments colored according to palette
- Clear labels with percentages
- Legend with brief descriptions

### Business Model Visualization
- Flow diagram showing relationships between:
  - Company
  - Customers
  - Revenue streams
  - Products/Services
- Simple iconography to represent different entities
- Directional arrows showing flow of money/value

### Financial Metrics
- Bar charts for year-over-year comparisons
- Line charts for trends over time
- Consistent color coding across all visualizations
